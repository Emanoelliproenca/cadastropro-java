# CadastroPro - Java 17

Aplicação de console desenvolvida em Java 17 para cadastro e gerenciamento de usuários.

O projeto foi desenvolvido com foco em consolidar fundamentos de lógica de programação, Programação Orientada a Objetos (POO) e manipulação de arquivos.

## Funcionalidades

- Cadastro de usuário (nome, email e idade)
- Validação de dados (email e idade)
- Bloqueio de email duplicado
- Listagem de usuários
- Busca por email
- Remoção por email ou nome
- Persistência de dados em arquivo (`usuarios.txt`)

## Tecnologias utilizadas

- Java 17
- IntelliJ IDEA

## Estrutura

- `Usuario.java` → Classe modelo
- `SistemaCadastro.java` → Regras de negócio e menu
- `Main.java` → Ponto de entrada da aplicação

## Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe `Main`
4. Utilize o menu no console
